CSS Questions

What is Selector's Specificity?

Selector's specificity is a measure that determines which CSS rule applies when multiple rules have different selectors pointing to the same element. It's calculated based on the types of selectors used, with a scoring system that ranks inline styles, IDs, classes, pseudo-classes, attributes, elements, and pseudo-elements in descending order of priority.

What are Complex Selectors?

Complex selectors are combinations of simple selectors, separated by combinators, that select elements based on their relationships to other elements. They can target child elements, siblings, or descendants, and are more specific than simple selectors.

Difference Between Pseudo-Classes and Pseudo-Elements

Pseudo-classes are used to define the style of an element in a specific state, such as when it is hovered over (:hover) or when it is active (:active). Pseudo-elements, on the other hand, allow you to style certain parts of an element, like the first line (::first-line) or before and after content (::before / ::after).
