Opening Developer Console:

  • In Mozilla Firefox:
  • Hotkeys: Ctrl + Shift + K.
  • Via Menu: Menu > Web Developer > Web Console.
  • In Safari:
  • Hotkeys: Command + Option + I.
  • Via Menu: Safari > Preferences > Advanced > Show Develop menu in menu bar.

Absolute vs. Relative Links:

Absolute links specify the full path to a resource, including the protocol and domain name. They are preferred when linking to an external resource or when the structure of the website may change.

Relative links specify the path relative to the current location of the file. They are convenient for links within the same website, as they simplify moving the site to another domain or directory.

Graphic Formats in Web Development:

  • JPG (or JPEG) - a lossy compression format, ideal for photographs with a wide range of colors.
  • GIF - supports animation and a limited palette of 256 colors, good for simple images and animations.
  • PNG - supports transparency and lossless compression, great for images with sharp edges, such as text and logos.
  • WEBP - a modern format developed by Google, offers both lossy and lossless compression, providing smaller file sizes at comparable quality.
  • SVG - a vector graphics format based on XML, allows images to be scaled without quality loss and supports interactivity and animation.
